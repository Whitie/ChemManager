Technik

Server

  • cross-platform (Windows/Linux/MacOS/*BSD)
  • Hauptsprache Python 3.x
  • Framework Django
  • Datenbank PostgreSQL (einfach austauschbar)
  • Webserver inklusive (CherryPy)
  • diverse Drittanbieter-Bibliotheken
  • alles Open-Source

Client

  • HTML / CSS 3 / Javascript
  • betriebssystemunabhängig, läuft in jedem modernen Browser
    • Chrome
    • Firefox
    • Opera
    • Internet Explorer > IE 8
  • läuft auch auf Mobilgeräten

Sonstiges

  • keine Limitierung bei der Anzahl der Clients
  • kann auch lokal am Einzelplatz betrieben werden
  • als Server für kleine Umgebungen (~10 Clients) reicht ein Raspberry Pi (~35€)
